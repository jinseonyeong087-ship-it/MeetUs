현재 동작

  • 업로드 후 상태가 `CREATED → UPLOADED → PROCESSING → COMPLETED` 순서로 변경됩니다.
  • 상세 화면에서는 처리 중 상태를 polling으로 재조회합니다.
  • 실패 상태 회의는 재처리 버튼을 통해 다시 진행할 수 있습니다.

목업 기준

  • 현재는 `localStorage` 기반 mock 데이터로 동작합니다.
  • API, DB 연동 후에도 동일한 UI 흐름을 유지하도록 설계했습니다.
