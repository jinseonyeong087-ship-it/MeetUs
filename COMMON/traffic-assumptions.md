# 트래픽 가정 문서 (AA)

## 초기 가정
- 워크스페이스 수: 5개
- 워크스페이스당 하루 회의 수: 3건
- 하루 총 회의 수: 15건
- 평균 회의 길이: 30분
- 동시 업로드 피크: 3~5건

## 처리량 추정
- 일 총 오디오 분량: 약 450분
- 월 총 회의 수: 약 450건
- 월 총 오디오 분량: 약 13,500분
- API 동시 요청 최대: 10건

## 용량 계획 기준
- 애플리케이션: ECS(Fargate) 단일 운영 환경 기준
- 데이터베이스: RDS `t3.micro` 기준 운영
- 자동 수평 확장(HPA): 미적용
- 파일 크기: 30MB 이하로 제한

## 운영 고려 항목
- 실제 워크스페이스별 사용 패턴
- 장기 보관 정책
- Transcribe 사용량 상한 임계치
